# Milesto 迭代式 PRD（个人任务管理开源替代品）

本文档作为 PRD 索引与迭代拆分入口。每个迭代 PRD 都应该是“可独立交付”的：清晰的目标、范围边界、验收标准与风险。

## 文档结构

- [docs/prd/00-overview.md](prd/00-overview.md)：产品概述（愿景、用户、原则、信息架构、数据模型、非目标）
- [docs/prd/01-v0.1-mvp.md](prd/01-v0.1-mvp.md)：v0.1 MVP（范围、优先级、验收、风险、里程碑）

阅读顺序建议: 先看 Overview，再看当前迭代版本 PRD（例如 v0.1 MVP）。

## 里程碑（建议）

- M0（1-2 周）: 基建与核心列表
- M1（2-4 周）: MVP 可用
- M2（4-6 周）: 功能增强
- M3（6-10 周）: 同步 PoC
